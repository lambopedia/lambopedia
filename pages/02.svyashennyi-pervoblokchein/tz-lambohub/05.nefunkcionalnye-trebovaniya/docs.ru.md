#### Интерфейс пользователя
Интерфейс пользователя должен быть мультиязычным. Первоначально должно быть три языка:
* русский;
* английский;
* эсперанто.  

Интерфейс пользователя должен корректно отображаться от 640 точек по ширине и высоте рабочей зоны окна страницы.
